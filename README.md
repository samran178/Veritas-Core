# Veritas Core // Real-Time AI Content Forensics Engine

Veritas Core is a high-performance, mobile-responsive text forensic analyzer dashboard engineered to detect automated text structures, linguistic anomalies, and machine-generated content patterns. Built to combat modern AI-generated spam and content noise, the platform delivers instant token-level and sentence-level forensic breakdowns entirely within the client layer.

## 🚀 Key Engineering Focus Areas

- **Cross-Device Fluidity:** 100% mobile-responsive interface utilizing Tailwind flex-grids, auto-collapsing analytics blocks, and accessible mobile touch-targets.
- **Linguistic Heuristic Engine:** Calculates text randomness (Perplexity Index), sentence length variance (Burstiness Rating), and structural word distribution (Repetition Index).
- **Interactive Highlight Interface:** Renders real-time text analysis overlays, isolating and highlighting robotic structures in neon coral/red and natural human prose in neon emerald.
- **Client-Side Scalability:** Built as a pure standalone frontend engine that avoids high backend computational compute costs by handling complex processing within web vectors.

## 🛠️ Built With

- **Framework:** React (Vite)
- **Styling UI:** Tailwind CSS (Dark-Mode Cyber Operations Aesthetic)
- **Data Visualization:** Recharts (Dynamic Integrity Metrics Charting)

## 📦 Architecture & Setup

### Local Installation
To spin up a local development instance of Veritas Core:

```bash
# 1. Clone the repository
git clone [https://github.com/samran178/Veritas-Core.git](https://github.com/samran178/Veritas-Core.git)

# 2. Navigate into the root directory
cd Veritas-Core

# 3. Install required node dependencies
npm install

# 4. Fire the local development server
npm run dev
